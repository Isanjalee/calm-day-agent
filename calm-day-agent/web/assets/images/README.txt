Place your chosen images in this folder using these filenames:

- hero-celestial.jpg
  Use for the top-right hero artwork.

- day-plan-art.jpg
  Use for the large art card under the day-plan form.

- diary-calm.jpg
  Use for the diary panel artwork.

- book-dreamscape.jpg
  Use for the book panel artwork.

The UI will automatically show the real images when these files exist.
If a file is missing, the app falls back to the built-in illustration style.
